
## Description

HTTP Server to download gouvernement csv, transform it and load it on a database for COVID19 purposes.
Servidor HTTP para descargar csv del gobierno, transformarlos y cargarlos en una base de datos para propositos de COVID19.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
